# ESCAPE FROM DEERISLE

By [Escape From Dayz](https://discord.gg/EscapeFromDayz "Escape From Dayz Discord")


> Escape From Dayz is a community which was founded in March 2022 and started doing Dayz Servers in July 2022. Our Servers take core ideas from games like Escape From Tarkov and brings them to DayZ. Please Join [Our Discord](https://discord.gg/EscapeFromDayz "Escape From Dayz Discord") To stay up to date on updates/announcements!

 ![alt text](https://github.com/Mitch3902/Escape-From-DeerIsle/blob/main/Escape_From_DayZ_DeerIsle.png?raw=true "Loading Screen")

## Server Description

ESCAPE FROM DEERISLE is a PvP DayZ server Based on Escape From Tarkov and uses many mods to achieve a similar but unique harcore experience, We also feature a extraction mod that will take you to a safezone where you can buy a Secure Stash and have access to traders from tarkov.

## Server Features

        - Escape From Tarkov Lore, Gear, and Items
        - Extractions to trader
        - BaseBuildingPlus
        - Factions
        - Secure Containers
        - Groups
        - First Person Only
        - Vehicle Third Person
        - Armored Vehicles
        - Dogtags
        - Keycards
        - Keycrates
        - AutoRun
        - Breaching Charge
        - Unlimited Stamina
        - Custom Map Edits
        - Airdrops
        - Custom Clothing
        - Custom HUD/UI
        - Ear Plugs
        - Helicopters
        - No Vehicle Damage
        - Tactical Flava
